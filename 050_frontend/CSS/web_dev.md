## position

- `static` - по умолчанию
- `relative` - можно сдвигать относит-но изначального положения
- `absolute` - сдвигается относ-но документа или родителя с `relative`
- `fixed` - сдвигается всегда относ-но браузера
- `sticky` - реагирует на скролинг
