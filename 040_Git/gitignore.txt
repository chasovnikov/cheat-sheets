
Файл .gitignore    - игнорирование файлов и каталогов по шаблонам:
#            - эта строка игнорируется
*.a          - не обрабатывать файлы, имя которых заканчивается на .a
!lib.a       - но отслеживать файл lib.a
/TODO        - игнорировать только файл TODO
build/       - игнорировать все файлы в каталоге build/
doc/*.txt    - игнорировать doc/notes.txt, но не doc/server/arch.txt
doc/**/*.txt - игнорировать все .txt файлы в каталоге doc/
             - Можно использовать стандартные glob шаблоны


Чтобы игнорировать файл, для которого ранее был сделан коммит, 
необходимо удалить этот файл из репозитория, а затем добавить 
для него правило в .gitignore. Используйте команду git rm с 
параметром --cached, чтобы удалить этот файл из репозитория, 
но оставить его в рабочем каталоге как игнорируемый файл.


    Игнорирование файлов через консоль (запись в файл .gitignore):
cat .gitignore
*.[oa]
*~
