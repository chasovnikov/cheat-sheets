id          — если вы хотите получить информацию о вашей учётной записи.
w           — какой пользователь сейчас находится в системе
-f          — для того, чтобы узнать, откуда был совершён вход
lsblk       — если вы хотите открыть список блочных устройств хранения данных
lscpu       — отображает информацию о процессорах
free        — показывает объём свободной и уже используемой памяти
free -g     -//-
lsb_release -a    — если вы хотите получить информацию о версии вашей ОС. 

ls          - показать содержимое текущего каталога
ls repos    - показать содержимое каталога "repos"
pwd         - (print working directory) полный путь до текущей директории
dd          - (dataset definition) делать копии разделов, перемещать их, удалять
rm          - (remove) для удаления директорий и их содержимого
grep        - полнотекстовый поиск в файле
-r          - рекурсивный поиск
-i          - игнорировать регистр (Case insensetive)
-l          - вывести результат списком
chown user123:group123     - изменить владельца файла
chmod       - изменяет права на файл или каталог
find        - искать файлы по определённым параметрам:
find / -name foo.txt -type f -print\
  -name       — искать по маске имени файла
  -delete     — удалять найденные файлы
  -empty      — только пустые каталоги
  -print      — показывает на экране найденные файлы
  -regex      — искать по имени файла, используя регулярные выражения
  -size       — размер
  -type — тип искомого: f=файл, d=каталог (directory), l=ссылка (link), p=канал (pipe), s=сокет
  и др.

cat file-name.txt         — показывает содержимое файлов
head file-name.txt        — показать первые 10 строк
head -n 4 file-name.txt   — показать первые 4 строки
  -n (--lines)      — заданное количество строк вместо 10
  -v (--verbose)    — перед текстом выводит название файла.
  и др.

touch     - создать новый файл или изменить время открытия папки/файла
wget      — загружает из интернета документы, картинки, целые страницы и прочий контент
wc        — считает количество строк, слов, знаков и байтов
zip       — упаковывает файлы в архив в формате .zip

strings     — команда выведет готовые к печати строки файла
od          — позволит вам напечатать файл в восьмеричном формат
cmp         — даёт возможность побайтно сравнивать файлы

comm      — отсортированные файлы будут строка за строкой
diff      — позволяет построчно выявить различия

  Команды для управления пользователями:
useradd     — создаёт новую учётную запись
passwd      — задаёт пароль для учётной записи
userdel     — удаляет выбранную учётную запись

  Для установки, удаления и обновления пакетов (Debian/Ubuntu):
sudo apt-get install название_пакета  — устанавливает программу на выбор
sudo apt remove название_пакета       — удалить не только ПО, но и все неиспользуемые пакеты
sudo apt-get update                   — обновляет списки пакетов, которые требуют обновления
sudo apt-upgrade                      — обновляет пакеты
sudo apt purge                        — удаляет все файлы конфигурации вместе с пакетами ПО
sudo apt autoremove -purge            — удаляет абсолютно всё: и пакеты, и зависимости, и конфиги

!!          — повторяет последнюю команду
!$          — позволяет изменить команду, сохраняя последний аргумент 
less !$     — повторное открытие файла в меньшем объёме
!*          — изменить команду, сохраняя все аргументы 
head states.txt | grep '^Al'      — при использовании должен быть хвост
tail !*         — избавляет от необходимости вводить остальную часть команды
>x.txt          — создание пустого файла или очистки существующего
lsof -P -i -n   — определяет, к каким скриптам идёт обращение со стороны веб-сервера
man             - (manual) для вывода полного руководства о той или иной команде
q               - (quit) выход из режима просмотра
