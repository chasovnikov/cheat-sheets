
• npm install

Проверить установку node и npm (узнать версии):
• node -v
• npm -v

Что бы устанавливать зависимости, нужно проинициализировать проект:
• npm init

Установить express:
• npm i express

Установить зависимость именно для разработки:
• npm i --save-dev nodemon

// Смотреть переменные окружения:
console.log(process.env);
console.log(process.PORT);

Чтобы прописывать переменные окружения в файле package.json установим пакет:
• npm i cross-env

Чтобы читать файл .env установим пакет
• npm i dotenv

// Ловить команды из консоли:
console.log(process.argv);

// Завершить процесс
process.exit();

Стандартн. модуль "path" позволяет удобно взаимодействовать с путями

Запустить nodemon (перезапускает сервер при изменениях в коде):
В package.json: 
"scripts": {
    "start": "nodemon index.js"
  },
В консоли:
• npm start
