
• npm install

Проверить установку node и npm:
• node -v     узнать версию
• npm -v

Что бы устанавливать зависимости, нужно проинициализировать проект:
• npm init
Далее ответить: название пакета (своего), его версию, описание, ...
В корне проекта должен появиться файл package.json

Запуск скрипта index.js из консоли:
• node index.js

Установить express:
• npm i express

Установить зависимость именно для разработки:
(будет использоваться только на этапе разработки)
• npm i --save-dev nodemon
