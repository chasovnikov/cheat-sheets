Node.js Рамки и инструменты
Node.js это низкоуровневая платформа. Для того, чтобы сделать работу простой и 
увлекательной для разработчиков, были созданы тысячи библиотек Node.js сообществом.

Многие из них со временем стали популярными вариантами. Вот неполный список тех, которые 
стоит изучить:

AdonisJS: Полнофункциональный фреймворк на основе машинописи, ориентированный на эргономику, 
стабильность и уверенность разработчиков. Адонис - один из самых быстрых Node.js веб-фреймворки.

Egg.js : Платформа для создания лучших корпоративных фреймворков и приложений с Node.js & Коа.

Express: Он предоставляет один из самых простых, но в то же время мощных способов создания 
веб-сервера. Его минималистский подход, непредвзятый, сосредоточенный на основных функциях сервера, 
является ключом к его успеху.

Fastify: веб-платформа, ориентированная на обеспечение наилучшего опыта разработчиков с наименьшими 
накладными расходами и мощной архитектурой плагинов. Fastify - один из самых быстрых Node.js 
веб-фреймворки.

FeatherJS: Feathers - это легкий веб-фреймворк для создания приложений реального времени и 
API-интерфейсов REST с использованием JavaScript или машинописи. Создавайте прототипы за 
считанные минуты, а готовые к производству приложения - за несколько дней.

Gatsby: Генератор статических сайтов на базе React, основанный на GraphQL, с очень богатой 
экосистемой плагинов и стартеров.

hapi: богатая платформа для создания приложений и сервисов, которая позволяет разработчикам 
сосредоточиться на написании логики приложений многократного использования вместо того, чтобы 
тратить время на создание инфраструктуры.

koa: Он создан той же командой, что и Express, и стремится быть еще проще и меньше, опираясь 
на многолетний опыт. Новый проект родился из необходимости внести несовместимые изменения, не 
нарушая существующего сообщества.

Loopback.io : Позволяет легко создавать современные приложения, требующие сложной интеграции.

Meteor: невероятно мощный фреймворк с полным стеком, предоставляющий вам изоморфный подход к 
созданию приложений с использованием JavaScript, совместному использованию кода на клиенте и сервере. 
Когда-то готовый инструмент, который предоставлял все, теперь интегрируется с интерфейсными 
библиотеками React, Vue и Angular. Может также использоваться для создания мобильных приложений.

Micro: Он предоставляет очень легкий сервер для создания асинхронных микросервисов HTTP.

NestJS: Прогрессивная программа на основе машинописи Node.js платформа для создания эффективных, 
надежных и масштабируемых серверных приложений корпоративного уровня.

Next.js : Платформа React, которая предоставляет вам лучший опыт разработчика со всеми функциями, 
необходимыми для производства: гибридный статический и серверный рендеринг, поддержка машинописи, 
интеллектуальное объединение, предварительная выборка маршрутов и многое другое.

Nx: Набор инструментов для разработки монорепо с полным стеком с использованием NestJS, Express, 
React, Angular и многого другого! Nx помогает масштабировать вашу разработку от одной команды, 
создающей одно приложение, до множества команд, работающих над несколькими приложениями!

Remix: Remix - это полнофункциональный веб-фреймворк для создания превосходного пользовательского 
интерфейса в Интернете. Он поставляется из коробки со всем необходимым для создания современных 
веб-приложений (как внешних, так и внутренних) и развертывания их в любой среде выполнения на основе 
JavaScript (включая Node.js ).

Sapper: Sapper - это платформа для создания веб-приложений всех размеров, с прекрасным опытом 
разработки и гибкой маршрутизацией на основе файловой системы. Предлагает SSR и многое другое!

Socket.io : Механизм связи в реальном времени для создания сетевых приложений.

Strapi: Strapi - это гибкая безголовая CMS с открытым исходным кодом, которая дает разработчикам 
свободу выбора их любимых инструментов и фреймворков, а также позволяет редакторам легко управлять 
и распространять свой контент. Делая панель администратора и API расширяемыми с помощью системы 
плагинов, Strapi позволяет крупнейшим компаниям мира ускорять доставку контента, создавая при 
этом прекрасные цифровые возможности.
